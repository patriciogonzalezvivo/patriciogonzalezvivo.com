Astros is a living, luminous meditation on the present sky — an interactive web piece that translates real-time celestial positions into a dynamic light sculpture that integrates symbolic glyphs with astronomical elements that slowly mirrors both the current sky and the symbolic horizon. As the cosmos shifts, so does the image: an ever-changing record of alignment rendered in light.

The work bridges ancient cosmology and contemporary computation. Drawing from live astronomical data, it renders the sky not as myth or memory but as a present condition — measured, translated, and momentarily held in form. What appears is not a chart but a contemplation: the cosmos as it is, right now, in this place.

Made in conversation with [Martin Bonadeo](https://martinbonadeo.art/), [Oliverio Duhalde](https://www.oliverioduhalde.com/) (sound design), and Alejandra Eusebi Polich (astrology).
